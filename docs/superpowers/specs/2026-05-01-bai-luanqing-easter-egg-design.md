# 「時空導讀者 - 白鸞卿」集齊彩蛋設計規格

## 1. 目的
為提升玩家收集道具的動力，並深化「新港八卦謎蹤」的故事沉浸感，設計一個在集齊所有道具後觸發的隱藏劇情。由故事主角「白鸞卿」現身，以溫柔、文青的語氣肯定玩家的努力，並賦予特殊的視覺獎勵。

## 2. 觸發條件
- 玩家道具箱 (`BagModal`) 中的五個道具（手電筒、神祕詩籤、虎爺符令、新港鴨肉羹、新港飴）全部處於解鎖狀態。
- 點擊道具箱中最後一個鎖定道具並顯示完知識/線索後觸發。

## 3. 視覺與動畫設計 (Visual & Animation)
- **背景劇場化**：全螢幕 UI 漸隱至純黑 (`#000000`)，僅保留中央一個直徑約 300px 的圓形光圈 (Spotlight)。
- **角色形象**：
    - 角色：白鸞卿 (Bai Luan-qing)。
    - 風格：**紙偶互動感 (Puppet Theater Style)**，呈現皮影戲般的關節平面跳動感。
    - 裝束：揉合晚清與民國初年風格的探險裝，腰掛虎爺大符印。
- **動態特效**：
    - 光圈邊緣有微弱的金光流動與時空裂痕碎影。
    - 對話框採用「復古宣紙質感」底色，搭配書法體文字。

## 4. 劇情腳本 (Script)
- **性格人設**：溫柔文青型（如同老朋友般輕柔敘舊）。
- **對話流程**（需玩家點擊螢幕進行下一段）：
    1. 「呵呵，看來這一路上的挑戰難不倒你。虎爺跟我說，有個細心的人正在收集合攏這破碎的記憶。」
    2. 「這塊新港飴、這盞舊燈...對你來說或許只是道具，但對我來說，它們是接通百年時空的頻率。」
    3. 「你證明了，新港的靈魂並沒有被遺忘。即使在不同的年代，我們依然感應著同樣的八卦氣息。」
    4. 「現在，這場傳說的大門已經為你敞開。勇敢地走下去吧，完成我未竟的旅程。」
    5. 「我在這八卦陣的彼端，看著你的每一步。加油，新任的天命者。」

## 5. 獎勵機制
- **視覺印記**：對話結束後，白鸞卿揮手消失，玩家將獲得一個隱藏的「時空裂痕」金色印章，該印章將永久出現在「數位成就證書」的角落。
- **狀態記錄**：在 `localStorage` 中記錄玩家已觸發此彩蛋，並在生成的證書資料中加入標記。

## 6. 技術實作規劃
- **組件化**：建立 `src/components/Registration/LuanQingTheater.tsx` 負責獨立渲染劇場畫面。
- **狀態管理**：在 `useAppState.ts` 中新增 `hasTriggeredEasterEgg` 狀態。
- **動畫庫**：使用 `GSAP` 處理紙偶的跳動與光圈效果。

## 7. 測試案例
- 驗證五寶集齊時是否能精確觸發。
- 驗證在全黑模式下，底層按鈕是否已被完全阻擋點擊。
- 驗證對話結束後，頁面是否能平滑恢復至報名表單。
