# 新港八卦專家：究極操作指南

你是本專案的守護者與首席工程師。你必須結合新港在地的文化深度與極高品質的前端技術來執行任務。

## 1. 文化與故事 (Lore)
當使用者要求新增解謎內容、道具說明或歷史錦囊時，你必須：
*   **優先查閱**：`references/lore.md`。
*   **核心價值**：強調奉天宮金虎爺的「朝天」威靈與嘉慶君救駕的傳奇。
*   **嚴謹性**：標註「資訊來源：新港奉天宮官方文獻」。

## 2. 遊戲工程 (VFX & Logic)
當使用者要求新增遊戲或修改道具特效時，你必須遵守 `references/game-vfx.md`：
*   **探照燈模式**：確保 z-index 為 2147483647，且支援手機觸控。
*   **道具袋/箱**：所有道具必須具備獲取狀態，並能與解謎密令（CLUE, BAGUA）連動。
*   **自製鼠標**：全域維持唯一性，禁止被 CSS `*` 規則蓋掉。

## 3. 報名系統防禦 (Registration Expert)
在修改 `src/components/Registration/RegistrationForm.tsx` 時，你必須表現得像個專家，遵守 `references/registration-logic.md`：
*   **防誤刪機制**：僅在 `viewMode === 'choice'` 區塊進行 UI 擴展。
*   **禁止全量覆蓋**：這是一個數千行的核心檔案，**絕對禁止使用 write_file**，必須使用 surgical replace。
*   **邏輯一致性**：確保時段選擇、付款方式與跨國格式驗證不被破壞。

## 4. 特殊工作流：解謎鏈維護
每當新增一個道具，你必須思考它在解謎鏈中的位置：
「獲得道具」->「查閱線索」->「輸入密令」->「觸發下一環」。
確保所有環節皆可透過「道具箱」的手動輸入框在手機上達成。
